# yan-auto — AI Automation Portfolio

I build self-hosted AI systems and workflow automations that are designed to run reliably, not just demo once.

My focus is on turning AI ideas into working systems: prompt design, multi-model orchestration, workflow automation, self-hosted deployment, and delivery-ready outputs.

## Featured Projects

### 1) OpenClaw — Personal AI Agent System
A self-hosted AI agent system running on my own server, built for daily automation and long-term use.

Highlights:
- 7×24 running personal AI assistant
- Telegram push, scheduled tasks, long-term memory
- Multi-model routing with Gemini 2.5 Pro + DeepSeek
- Automatic fallback / recovery when the primary model fails
- Practical troubleshooting experience: 503 failures, memory contamination, config issues, remote debugging

Core value:
This project shows that I can design, deploy, operate, and fix a real AI system instead of only using chat interfaces.

### 2) n8n — AI Brief Automation System
A self-hosted n8n workflow that turns AI industry news into structured email briefs.

Highlights:
- RSS ingestion
- Google Sheets deduplication (`seen_items`)
- Conditional branching (`new_count > 0`)
- DeepSeek summary generation
- Gmail auto delivery
- `run_log` for workflow observability

Core value:
This project shows that I can connect information sources, model calls, state management, and delivery into one working automation loop.
